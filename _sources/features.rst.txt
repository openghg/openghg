========
Features
========

OpenGHG is a platform that makes the retrieval and manipulation of greenhouse gas data easy. All data on the platform is processed
and stored in a CF compliant format with all associated metadata attached. This makes it easy to retrieve data, make comparisons
and run simulations.

- Standardised format
- Easy retrieval of set datasets
- All associated metadata attached
- CF Compliant format
- Merge time series data with footprints
- Retrieve emissions / flux data
- Retrieve and use meteorological data
