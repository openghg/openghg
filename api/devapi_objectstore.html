Objectstore#

These functions are used by OpenGHG to store and retrieve data from the object store. OpenGHG will call either local object store functions (that create an object store on disk) or cloud object store functions depending on its environment. Within the __init__.py of openghg.objectstore a check is made to see if the environment is the OpenGHG hub or a serverless function.

Local#

These functions handle data storage on disk.
