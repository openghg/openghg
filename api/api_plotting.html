Plotting#

These functions help in the creation of simple plots.

openghg.plotting.plot_timeseries(data, xvar=None, yvar=None, title=None, xlabel=None, ylabel=None, units=None, logo=True, calibration_scale=None)[source]#

Plot a timeseries

Parameters:

  • data (ObsData | ObsColumnData | list[ObsData | ObsColumnData]) – ObsData object or list of objects

  • xvar (Optional[str]) – x axis variable, defaults to time

  • yvar (Optional[str]) – y axis variable, defaults to species

  • title (Optional[str]) – Title for figure

  • xlabel (Optional[str]) – Label for x axis

  • ylabel (Optional[str]) – Label for y axis

  • units (Optional[str]) – Units for y axis

  • logo (bool | None) – Show the OpenGHG logo

  • calibration_scale (Optional[str]) – Convert to this calibration scale

Returns:

Plotly Graph Object Figure

Return type:

go.Figure
