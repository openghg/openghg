JobRunner¶

The JobRunner class is used to run jobs on HPC clusters.

class openghg.client.JobRunner(service_url)[source]¶

An interface to the the jobrunner service on the OpenGHG platform.

This class is used to run jobs on both local and cloud based HPC clusters

Process¶

The Process class is used upload data for processing by OpenGHG.

Search (Client)¶

The Search class is used to search for data in the object store.

RankSources¶

The RankSources class is used to rank sources of data. Setting a high rank for a Datasource across a specific daterange means that data from that Datasource will be preferred when a user searches for data.

Retrieve¶

The Retrieve class is used to retrieve data from the object store.
