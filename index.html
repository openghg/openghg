OpenGHG#

OpenGHG is a project to create a platform for collaboration for researchers, stakeholders and the public within the realm of greenhouse gases. The platform is built on open source technologies technologies to create a cloud agnostic platform for simulation and analysis of large datasets.

Background#

OpenGHG is based on the work of the HUGS project to develop a cloud based platform for analysis of greenhouse gas measurement data.
